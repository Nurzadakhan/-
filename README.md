# Diploma Generator (Генератор дипломов и приложений)

Автоматическая генерация дипломов и вкладышей (приложений) на основе Excel-данных.  
Готовый `.exe` не требует установки Python и работает на Windows.

## Возможности

- Загрузка шаблонов дипломов (PNG) и наложение текста из Excel.
- Поддержка казахского и русского языков.
- Автоматическая нумерация дисциплин и разбивка таблиц на несколько страниц.
- Опция «Дубликат»: курсивная надпись в правом верхнем углу.
- Графический интерфейс на Tkinter с прогресс-баром.
- Сохранение результата в PDF (A4 landscape).
- Генерация шаблона Excel с правильными колонками (кнопка «Скачать шаблон»).
- Упаковка всех ресурсов (шаблоны, шрифты, логотип) в `.exe` через PyInstaller.
